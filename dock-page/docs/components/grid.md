# 栅格组件 Row & Col

> 栅格组件用于提供基本的布局能力

```html
<c-row :gutter="10">
  <c-col :span="8">common-vue-ui</c-col>
  <c-col :span="8">common-vue-ui</c-col>
  <c-col :span="8">common-vue-ui</c-col>
  <c-col :span="3" :pull="3">common-vue-ui</c-col>
  <c-col :span="3" :push="2">common-vue-ui</c-col>
</c-row>
```
