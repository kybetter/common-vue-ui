common-vue-ui

一款面向 C 端的 vue 组件库

快速上手 →

兼容 IE

因为需要面向 C 端用户，所以组件库在设计上尽量兼容 IE9 以上浏览器。组件库适配 vue2 版本。

自定义组件样式

大部分组件提供了默认样式，你可以很方便的覆盖它以便自定义你的样式。

工具库

不仅提供了组件，还提供了一些常用的 css 样式工具，使用这些工具样式可以节省开发时间。
